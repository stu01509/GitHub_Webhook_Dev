# GitHub Webhook Dev

## Usage

1. Clone the Repo

    ```sh
    git clone https://github.com/stu01509/GitHub_Webhook_Dev.git
    ```

2. Install the NPM dependencies

    ```sh
    cd GitHub_Webhook_Dev
    yarn
    ```

3. Start the Application

    ```sh
    WEBHOOK_SECRET=YOUR_WEBHOOK_SECRET npm run start  
    ```
